# Eksperimen_SML_Nama-siswa

Repository ini disiapkan untuk memenuhi Kriteria 1 submission SMSML.

Isi repository:
- `namadataset_raw/credit_scoring_raw.csv`
- `preprocessing/Eksperimen_Nama-siswa.ipynb`
- `preprocessing/automate_Nama-siswa.py`
- `preprocessing/namadataset_preprocessing/credit_scoring_preprocessing.csv`
- `.github/workflows/preprocess.yml`

Langkah pakai:
1. Upload folder ini ke repository GitHub public bernama `Eksperimen_SML_Nama-siswa`.
2. Jalankan notebook eksperimen.
3. Pastikan file preprocessing hasil output tersedia.
4. Copy URL repo public ke file `Eksperimen_SML_Nama-siswa.txt` pada submission ZIP.
